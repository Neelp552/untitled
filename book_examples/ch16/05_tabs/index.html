Today, more than half of the 10,000 most-visited websites use jQuery, and that number is growing every day. In fact, jQuery has joined HTML, CSS, and JavaScript as an essential part of every web developer’s resume.

That's where this book comes in. Fast-paced, professional, and packed with real-world examples, it delivers the jQuery expertise you need...and the JavaScript knowledge you need to tweak jQuery code for specific applications. That's true whether you use it for step-by-step training or for on-the-job reference.

Zak Ruvalcaba has been researching, designing, and developing for the Web since 1995. He holds a Bachelor's degree from San Diego State University and a Master of Science degree in instructional technology from National University in San Diego.

Zak's skillset includes HTML/HTML5, CSS/CSS3, JavaScript, jQuery, ASP.NET, ADO.NET, Visual Basic, C#, Web Services, and Flash/ActionScript. He is also a Microsoft Certified Application Developer for .NET (MCAD) and a Microsoft Certified Solutions Developer for .NET (MCSD).

Besides working as a web developer, Zak is currently the Lead Instructor for the Front-End Web Developer Certificate Program in the San Diego Community College District. He is not only responsible for developing the curriculum, but also hiring adjunct instructors to help teach the seven courses in this program.

Zak co-authored Murach's HTML5 and CSS3 by Mike Murach & Associates, and he is the lead author on this book.

Due to our unique presentation methods and this book's modular organization, this is the right book for any web developer who wants to use JavaScript effectively. That includes:

  • Web developers who know HTML5 and CSS3 and are ready to master JavaScript and jQuery.
  • Web developers who program in ASP.NET, JSP, or PHP on the server side and now want to master client-side programming.
  • Web developers who have already read 3 or 4 JavaScript or jQuery books but still don't know how to get the most from jQuery UI and jQuery plugins.
